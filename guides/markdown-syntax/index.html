    Markdown Syntax Guide

    Learn how to format your documentation with Markdown.

    Headings

    # Heading 1
## Heading 2
### Heading 3
#### Heading 4
##### Heading 5
###### Heading 6

    Text Formatting

    Bold text - wrap with **text**

    Italic text - wrap with *text*

    Bold and italic - wrap with ***text***

    ~~Strikethrough~~ - wrap with ~~text~~

    Lists

    Unordered Lists

    • Item 1
    • Item 2
    • Nested item 2.1
    • Nested item 2.2
    • Item 3

    Ordered Lists

    1. First item
    2. Second item
    3. Nested item 2.1
    4. Nested item 2.2
    5. Third item
